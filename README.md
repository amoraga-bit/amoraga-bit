### 📚 Mi Enfoque Educativo

Utilizo **GitHub Classroom** como la columna vertebral de mis cursos para simular un entorno de desarrollo profesional.

* **Pizarras Educativas:** Clases visuales que conectan la teoría (diagramas de flujo) con la práctica (código), como se ve en mi banner.
* **Aprender Haciendo:** Proyectos prácticos y asignaciones basadas en repositorios.
* **Colaboración:** Fomento el uso de Pull Requests y revisiones de código entre compañeros.
* **Automatización:** Introducción básica a CI/CD para validar la calidad del código de los estudiantes.
